# 📖 Overview

SkillBridge is a modern web platform designed to make learning more accessible through **peer-to-peer knowledge exchange**.

Instead of relying solely on paid courses, users can connect with others who possess different skills, explore learning opportunities, join a collaborative community, and access curated educational resources.

The project focuses on creating an engaging, responsive, and user-friendly experience while promoting lifelong learning.
